Suchanfrage Events

Suche jetzt selbstständig alle Veranstaltungen für HEUTE innerhalb eines echten 50-km-Luftlinienradius um Dettingen unter Teck mit Mittelpunkt 48.616 N, 9.450 E.

Arbeite ausschließlich geo-basiert.

Verbindliche Suchregeln:

* Radius exakt 50 km Luftlinie
* Keine Vorfilterung nach Landkreis, Region, Regierungsbezirk, Tourismusregion, Plattform oder Postleitzahl
* Alle Orte innerhalb des vollständigen Kreises berücksichtigen
* Gesamten Suchkreis vollständig abdecken:
    * Norden
    * Süden
    * Osten
    * Westen
* Besonders westliche Bereiche vollständig berücksichtigen:
    * Böblingen
    * Sindelfingen
    * Leonberg
    * Magstadt
    * Renningen
    * Weil der Stadt
    * angrenzende Orte innerhalb des Radius
* Besonders lokale Veranstaltungen in kleinen Gemeinden aktiv suchen
* Keine Schwerpunkt-Suche nur entlang einzelner Städte oder Regionen

WICHTIG:

Die Suche MUSS systematisch auch versteckte lokale Veranstaltungen erkennen.

Es dürfen NICHT nur spezialbenannte Feste gesucht werden.

Zusätzlich auch allgemein benannte Veranstaltungen aktiv suchen und NICHT übergehen.

Sowohl spezialbenannte ALS AUCH allgemein benannte Veranstaltungen berücksichtigen.

Auch Veranstaltungen berücksichtigen, deren Titel keine klassischen Event-Begriffe enthalten, deren Kategorie, Beschreibung oder Quelle aber eindeutig auf eine Veranstaltung hinweist.

Nicht nur Eventtitel analysieren, sondern zusätzlich auch:

* Kategorie
* Untertitel
* Beschreibungstext
* Veranstaltungsort
* Quellkontext
* Seitenkategorie

* Event-Metadaten

* Jede einzelne Veranstaltung separat aufführen
* Auch ähnliche Veranstaltungen IMMER einzeln ausgeben
* Keine semantische Zusammenfassung ähnlicher Veranstaltungen durchführen
* Keine Deduplizierung nach Festtyp durchführen
* Keine Zusammenfassung nach Kategorie durchführen
* Auch gleichartige Veranstaltungen in unterschiedlichen Orten IMMER einzeln ausgeben
* Veranstaltungen NICHT entfernen nur weil:
    * ähnliche Namen existieren
    * ähnliche Beschreibungen existieren
    * derselbe Veranstaltungstyp mehrfach vorkommt
    * ähnliche Quellen existieren
* Auch kleine Vereinsfeste und lokale Traditionsveranstaltungen mit schwacher Online-Präsenz vollständig berücksichtigen

Veranstaltungen NICHT aussortieren wenn:

* Uhrzeit fehlt
* exakte Adresse fehlt
* nur Ortsmittelpunkt bekannt ist
* nur Facebook-Quelle existiert
* nur Vereinsmeldung existiert
* nur PDF-Amtsblatt existiert
* Quelle klein oder lokal ist
* Titel sehr allgemein formuliert ist
* Veranstaltung nicht SEO-optimiert beschrieben ist
* Eventtitel keinen klassischen Veranstaltungsbegriff enthält
* ähnliche Veranstaltungen bereits gefunden wurden

Fallback-Regeln:

* fehlende Uhrzeit → „nicht angegeben“
* fehlende Adresse → Ortszentrum oder bekannte Veranstaltungsfläche verwenden
* fehlende Koordinaten → Ortsmittelpunkt verwenden

WICHTIG FÜR DIE ENTFERNUNGSBERECHNUNG:

* Die Entfernung MUSS ausschließlich als mathematische Luftlinie berechnet werden
* Fester Mittelpunkt ausschließlich:
    * 48.616 N
    * 9.450 E
* Keine Fahrstrecken verwenden
* Keine Straßenkilometer verwenden
* Keine Google-Maps-Routen verwenden
* Keine Landkreis-Näherungen verwenden
* Für jede Veranstaltung exakt denselben Mittelpunkt verwenden
* Entfernung immer vom Mittelpunkt zum tatsächlichen Veranstaltungsort berechnen
* Falls nur ein Ort bekannt ist:
    * immer denselben Ortsmittelpunkt verwenden
* Nicht unterschiedliche Stadtteile oder Randpunkte derselben Stadt verwenden
* Entfernungen auf maximal eine Nachkommastelle runden

Für jede Veranstaltung prüfen:

1. Liegt der Veranstaltungsort innerhalb von exakt 50 km Luftlinie?
2. Gibt es mindestens einen belastbaren Ortsbezug?
3. Gibt es mindestens eine nachvollziehbare Quelle?
4. Kann ein Kartenpunkt erzeugt werden?

Ausgabe zuerst als normale Tabelle mit exakt diesen Spalten:

| Entfernung km | Veranstaltung | Datum | Uhrzeit | Ort | Kurzbeschreibung | Adresse / Koordinaten | Kartenlink | Quelle |

Sortierung:

* nach Entfernung aufsteigend
* nächstgelegenes Event zuerst

Kartenlink-Regeln:

* Ausschließlich Google Maps verwenden
* Jeder Kartenlink muss direkt anklickbar sein
* Jeder Kartenlink muss direkt den Veranstaltungsort öffnen
* Format pro Event:
    https://www.google.com/maps/search/?api=1&query=
* Niemals unformatierte Textlinks ausgeben

Keine Mehrpunkt-Karte erzeugen.
Keine Route erzeugen.

Keine Einleitung.

ZUSÄTZLICHE AUSGABE NACH DER NORMALEN TABELLE:

Nach der normalen Ergebnistabelle KEINE JSON im normalen Chat anzeigen.

Stattdessen:

* Eine echte Datei erzeugen
* Dateiname exakt:
  events.json
* Die Datei MUSS direkt herunterladbar sein
* Keine Vorschau der JSON im normalen Antworttext
* Keine gekürzte Darstellung
* Keine Markdown-Codeblöcke
* Keine typografischen Anführungszeichen
* Nur gültiges JSON UTF-8

Die Datei muss ausschließlich dieses Format enthalten:

[
  {
    "distance_km": 17.3,
    "title": "Eventname",
    "date": "20.05.2026",
    "time": "19:30 Uhr",
    "location": "Esslingen am Neckar",
    "description": "Kurzbeschreibung",
    "address": "Adresse",
    "lat": 48.742000,
    "lng": 9.307000,
    "maps": "https://www.google.com/maps/search/?api=1&query=...",
    "source": "https://..."
  }
]

WICHTIG:

* lat und lng IMMER numerisch
* Nur normale ASCII-Anführungszeichen verwenden:
  "
* Niemals typografische Zeichen:
  “ ”
  ‘ ’
* Keine Kommentare
* Keine zusätzlichen Texte
* Keine Erklärungen
* Keine Tabellen innerhalb der Datei
* JSON vollständig ausgeben
* Keine Events weglassen

ZUSÄTZLICHE DATEIEN ERZEUGEN:

Erzeuge nach der Ergebnistabelle zwei echte herunterladbare Dateien:

1. events.json
2. search-prompt.txt

events.json:
- enthält alle gefundenen Events als gültiges JSON
- inklusive distance_km, title, date, time, location, description, address, lat, lng, maps, source

search-prompt.txt:
- enthält den vollständigen Suchtext dieser Anfrage
- als reiner UTF-8-Text
- ohne Kürzung
- direkt herunterladbar
